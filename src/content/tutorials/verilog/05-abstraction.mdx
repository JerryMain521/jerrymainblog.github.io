## 以 4 选 1 数据选择器为例

4 选 1 数据选择器根据两位控制信号 `cntrl1`、`cntrl2` 从四路输入中选择一路输出。

<div class="not-prose">

| cntrl1 | cntrl2 | out |
|:------:|:------:|:---:|
| 0 | 0 | in1 |
| 0 | 1 | in2 |
| 1 | 0 | in3 |
| 1 | 1 | in4 |

</div>

> 逻辑表达式：`out = ~c1·~c2·in1 + ~c1·c2·in2 + c1·~c2·in3 + c1·c2·in4`

## 6.1 门级描述

直接用**逻辑门元件**例化来描述电路结构，需要先画出逻辑电路图。

<VerilogModule
  code={`// 4 选 1 MUX — 门级描述
module mymux (out, in1, in2, in3, in4, cntrl1, cntrl2);
  output out;
  input  in1, in2, in3, in4, cntrl1, cntrl2;
  wire   notcntrl1, notcntrl2, w, x, y, z;

  not (notcntrl1, cntrl1);          // 对控制信号取反
  not (notcntrl2, cntrl2);
  and (w, in1, notcntrl1, notcntrl2); // 00 → in1
  and (x, in2, notcntrl1, cntrl2);   // 01 → in2
  and (y, in3, cntrl1,    notcntrl2); // 10 → in3
  and (z, in4, cntrl1,    cntrl2);   // 11 → in4
  or  (out, w, x, y, z);             // 四路或
endmodule`}
  annotations={[
    { line: 7,  label: "NOT 门", desc: "内置门元件：not(输出, 输入)" },
    { line: 9,  label: "AND 门", desc: "and(输出, 输入1, 输入2, 输入3)" },
    { line: 13, label: "OR 门",  desc: "or(输出, 多路输入)" },
  ]}
/>

**特点：** 最接近电路底层，可读性较低，但综合结果可预测。

## 6.2 RTL 级描述

用**逻辑表达式**（`assign`）描述数据流，需先写出逻辑函数：

<VerilogModule
  code={`// 4 选 1 MUX — RTL 级（数据流）描述
module mux4_1 (out, in1, in2, in3, in4, cntrl1, cntrl2);
  output out;
  input  in1, in2, in3, in4, cntrl1, cntrl2;

  assign out = (in1 & ~cntrl1 & ~cntrl2) |
               (in2 & ~cntrl1 &  cntrl2) |
               (in3 &  cntrl1 & ~cntrl2) |
               (in4 &  cntrl1 &  cntrl2);
endmodule`}
  annotations={[
    { line: 6,  label: "assign", desc: "连续赋值，out 随输入实时更新" },
    { line: 6,  label: "逻辑表达式", desc: "直接对应真值表的最小项之和" },
  ]}
/>

**特点：** 比门级更简洁，直接对应布尔表达式，综合效率高。

## 6.3 行为级描述

只需知道输入输出的**真值表关系**，无需关心电路内部结构。

### 用 case 语句

<VerilogModule
  code={`// 4 选 1 MUX — 行为级（case 语句）
module mux4_1 (out, in1, in2, in3, in4, cntrl1, cntrl2);
  output out;
  input  in1, in2, in3, in4, cntrl1, cntrl2;
  reg out;

  always @(in1 or in2 or in3 or in4 or cntrl1 or cntrl2)
    case ({cntrl1, cntrl2})  // 拼接两位控制信号
      2'b00: out = in1;
      2'b01: out = in2;
      2'b10: out = in3;
      2'b11: out = in4;
      default: out = 1'bx;
    endcase
endmodule`}
  annotations={[
    { line: 7, label: "敏感列表", desc: "所有输入信号均列出，保证组合逻辑正确触发" },
    { line: 8, label: "位拼接", desc: "{cntrl1, cntrl2} 拼成 2 位选择信号" },
    { line: 13, label: "default", desc: "防止生成隐含锁存器" },
  ]}
/>

### 用条件运算符（最简洁）

```verilog
module mux4_1 (out, in1, in2, in3, in4, cntrl1, cntrl2);
  output out;
  input  in1, in2, in3, in4, cntrl1, cntrl2;

  assign out = cntrl1 ? (cntrl2 ? in4 : in3)
                      : (cntrl2 ? in2 : in1);
endmodule
```

## 三种描述方式对比

| 对比项 | 门级 | RTL 级 | 行为级 |
|--------|------|--------|--------|
| 需要了解电路结构 | ✅ 需画逻辑图 | ✅ 需写逻辑表达式 | ❌ 只需知道功能 |
| 代码可读性 | 低 | 中 | 高 |
| 综合结果可控性 | 高 | 中 | 低（由综合器决定） |
| 抽象层次 | 低 | 中 | 高 |
| 典型语句 | `and`、`or`、`not` | `assign` | `always` + `case`/`if` |
