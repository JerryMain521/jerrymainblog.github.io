## 运算符分类

按**功能**分为 9 类，按**操作数个数**分为 3 类：

| 类别 | 运算符 | 操作数 |
|------|--------|--------|
| 算术运算符 | `+` `-` `*` `/` `%` | 双目 |
| 逻辑运算符 | `&&` `\|\|` `!` | 双目 / 单目 |
| 位运算符 | `~` `&` `\|` `^` `^~` | 单目 / 双目 |
| 关系运算符 | `<` `<=` `>` `>=` | 双目 |
| 等式运算符 | `==` `!=` `===` `!==` | 双目 |
| 缩减运算符 | `&` `~&` `\|` `~\|` `^` `^~` | 单目 |
| 移位运算符 | `<<` `>>` | 单目 |
| 条件运算符 | `? :` | 三目 |
| 位拼接运算符 | `{ }` | — |

---

## 算术运算符

```verilog
5 + 3    // 加法 → 8
5 - 3    // 减法 → 2
5 * 3    // 乘法 → 15
7 / 2    // 整除（舍去小数）→ 3
-11 % 3  // 求模，符号取第一个操作数 → -2
```

> 若某操作数为不定值 `x`，则整个结果也为 `x`。

---

## 逻辑运算符

逻辑运算符把操作数视为**布尔值**，结果为 1 位（`0` / `1` / `x`）：

| 操作数 | 被视为 |
|--------|--------|
| 非零值 | 真（1） |
| 零 | 假（0） |
| `4'bxx00` | 不确定（x，可能为零） |
| `4'bxx11` | 真（1，一定非零） |

```verilog
a && b   // 逻辑与
a || b   // 逻辑或
!a       // 逻辑非（优先级最高）
```

---

## 位运算符

对操作数**逐位运算**，结果与操作数位数相同：

```verilog
~A          // 按位取反
A & B       // 按位与
A | B       // 按位或
A ^ B       // 按位异或
A ^~ B      // 按位同或（XNOR）
```

**位数不同时**，自动右端对齐，位数少的高位补 0：

```verilog
// A = 5'b11001, B = 3'b101
// → B 补齐为 5'b00101
A & B  // = 5'b00001
```

---

## 关系运算符

结果为 1 位逻辑值（`0` / `1` / `x`）：

```verilog
a < b    // 小于
a <= b   // 小于等于
a > b    // 大于
a >= b   // 大于等于
```

> 若某操作数为 `x`，则结果为 `x`。  
> 关系运算符优先级**低于**算术运算符。

## 等式运算符

| 运算符 | 说明 | 结果含 x？ |
|--------|------|-----------|
| `==` | 逐位相等（x/z 位结果为 x） | 会 |
| `!=` | 不等于 | 会 |
| `===` | 全等（包含 x、z 的精确比较） | 不会 |
| `!==` | 不全等 | 不会 |

```verilog
4'b1x0z == 4'b1x0z   // → x（不确定）
4'b1x0z === 4'b1x0z  // → 1（完全匹配，包括 x 和 z）
```

> `===` 和 `!==` 常用于 `case` 表达式，又称 **case 等式运算符**。  
> ⚠️ MAX+PLUS II 和 Quartus II 不支持 `===` / `!==`。

## 缩减运算符

**单目运算符**，对单个操作数**递推运算**，最终缩减为 1 位：

```verilog
reg [3:0] a = 4'b1010;
b = &a;   // = ((a[0]&a[1])&a[2])&a[3] → 0
b = |a;   // = ((a[0]|a[1])|a[2])|a[3] → 1
b = ^a;   // 所有位异或
```

**三类运算符对比（A = 4'b1010，B = 4'b1111）：**

| 类型 | 运算 | 结果 |
|------|------|------|
| 按位运算 | `A & B` | `4'b1010` |
| 缩减运算 | `&A` | `1'b0`（逐位 AND 缩减） |
| 逻辑运算 | `A && B` | `1'b1`（A、B 均为真） |

## 移位运算符

```verilog
4'b1001 >> 3   // 右移 3 位 → 4'b0001
4'b1001 >> 4   // 右移 4 位 → 4'b0000（移出全丢）
4'b1001 << 1   // 左移 1 位 → 5'b10010（位宽扩充）
4'b1001 << 2   // 左移 2 位 → 6'b100100
1 << 6         // → 32'b1000000
```

> 移 n 位 ≡ 乘以或除以 2ⁿ；右移数据丢失，左移自动扩展位宽。

---

## 条件运算符

**三目运算符**，语法：`信号 = 条件 ? 表达式1 : 表达式2`

```verilog
// 2 选 1 数据选择器
assign out = sel ? in1 : in0;
// sel=1 → out=in1；sel=0 → out=in0

// 三态门
assign out = enable ? in : 'bz;

// 嵌套实现 4 选 1
assign out = cntrl1 ? (cntrl2 ? in4 : in3)
                    : (cntrl2 ? in2 : in1);
```

## 位拼接运算符

将多个信号的某些位**拼接**成一个整体：

```verilog
{a, b[3:0], w, 3'b101}
// = {a, b[3], b[2], b[1], b[0], w, 1'b1, 1'b0, 1'b1}

{cout, sum} = a + b + cin;   // 9 位加法结果
{4{w}}                       // = {w, w, w, w}（重复法）
{b, {3{a, b}}}              // = {b, a, b, a, b, a, b}（嵌套）
```
